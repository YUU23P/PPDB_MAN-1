Program & Fasilitas Unggulan

Kami menyediakan berbagai program dan fasilitas pendukung untuk mengembangkan potensi siswa secara optimal.

Program Tahfiz

Program menghafal Al-Qur'an dengan target hafalan minimal 3 juz selama masa pendidikan.

  • Metode pembelajaran efektif
  • Pembina dari hafiz berpengalaman
  • Program murajaah berkala

IT & Coding Club

Pengembangan kemampuan teknologi informasi dan pemrograman untuk mempersiapkan siswa di era digital.

  • Pembelajaran coding dasar
  • Pengembangan aplikasi sederhana
  • Persiapan kompetisi IT

Program Bahasa

Penguasaan bahasa Arab dan Inggris dengan pendekatan komunikatif dan praktis.

  • Native speaker session
  • Language camp tahunan
  • Kelas persiapan TOEFL
